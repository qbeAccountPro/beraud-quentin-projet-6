A1 : App PayMyBuddy
This application facilitate the transfer of money between buddy. 

A2 : What you need to install 
- Java 20
- apache Maven 4.0.0
- Spring Boot 3.0.10

A3 : Password for UI testing :
Each password in UI testing is the "firstName" + "." + "lastName".

Exemple : 

Data : John, Doe with john.doe@gmail.com...

Connexion :
	Username : john.doe@gmail.com
	Password : john.doe

They are two application properties. One dedicace to prod and one another to testing.
