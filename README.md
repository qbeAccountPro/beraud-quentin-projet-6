# App PayMyBuddy
This application facilitate the transfer of money between buddy. 

## What you need to install 
- Java 20

## Password for UI testing :
Each password in UI testing is the "Name" + "." + "Surname".

### Exemple : 
Name : John
Surname : Doe
Mail : john.doe@gmail.com
ect...
